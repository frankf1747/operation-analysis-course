必背公式(今晚反复练习)

  1. $F(Q^*) = c_u / (c_u + c_o) = (p-c)/p$
  2. 投标价格决策:$r_j \ge \pi$? 接受 : 拒绝
  3. 流量守恒:$\sum_j x_{ji} + b_i = \sum_j x_{ij}$,每个节点都成立
  4. 三角分布 CDF:当 $x \le b$ 时 $F(x) = (x-a)^2 / [(b-a)(c-a)]$;当 $x > b$ 时 $1 - (c-x)^2/[(c-b)(c-a)]$
  5. 经验分位数:排序后第 ⌈$n\alpha$⌉ 个值
  6. TSP 子环路消除约束(SEC):$\sum_{i \in S} \sum_{j \in S, j \ne i} x_{ij} \le |S| - 1$
  7. 选择概率:对所有以商品 $i$ 为首选的分群 $\lambda_k$ 求和
  8. 定制化上界:$\sum_k \lambda_k \max_{i: u_{k,i} > u_{k,0}} r_i$
  9. 定价需求函数:$d_i = \alpha_i - \beta_i p_i + \sum_{j \ne i} \gamma_{ij} p_j$

必备决策流程

问题流程
"原问题是否最优?"(a) 原问题可行?(b) 对偶问题可行?(c) 两者目标函数值相等?若三者皆是 → 最优。若对偶不可行 → 无法得出结论。
"是否应该多买 X?"比较单位成本与影子价格。当且仅当成本 < 影子价格时购买。
"若资源变化 $\Delta$,收益变化多少?"$\Delta R \approx p^* \cdot \Delta b$。
"商品组合收益的上界?"(a) 定制化上界:$\sum \lambda_k \max r_i$,对相关商品取最大。或 (b) LP 松弛。
"TSP 为什么需要 SEC?"仅凭度数约束会允许出现不连通的子环路(小循环)。
"提供全部商品是否最优?"通常不是。存在同类相食(cannibalization)。需要整数规划。
"是否接受这个收益管理请求?"计算(或使用给定的)投标价格 $\pi$。当且仅当票价 ≥ $\pi$ 且容量充足时接受。
"如何计算报童问题的 $Q^*$?"(1) 计算 $\alpha$,(2) 识别分布,(3) 求 $F$ 的反函数。

快速合理性检查

  • 报童问题中 $\alpha < 0.5$ → $Q^* < $ 中位数(少订)。$\alpha > 0.5$ → $Q^* > $ 中位数。
  • 影子价格 = 0 → 约束存在松弛(资源未完全使用)。
  • 影子价格 > 0 → 约束紧绷(资源完全使用)。
  • 最大可靠性:$r < 1$ 的对数为负数。最大值 = 最接近零的负数之和。
  • TSP SEC 必须同时包含 $x_{ij}$ 和 $x_{ji}$。
  • 定价:$\beta > 0$ 恒成立;自身价格项贡献 $-\beta p^2$(为减号)。
  • 选择概率:argmax 中必须始终包含无购买选项。
  • 网络:$b_i$ 之和必须为 0(平衡)。

三角分布快速参考

Tr$[a, b, c]$: a=最小值,b=众数,c=最大值。

反 CDF:

  • $F(b) = (b-a)/(c-a)$ — 用以判断在哪一段。
  • 若 $\alpha \le F(b)$:$Q^* = a + \sqrt{\alpha(b-a)(c-a)}$
  • 若 $\alpha > F(b)$:$Q^* = c - \sqrt{(1-\alpha)(c-b)(c-a)}$

均值: $(a+b+c)/3$。
