# 第三周 —— 报童问题 Newsvendor Problem

## 0. 总览

报童以成本 $c$ 购入当天的报纸,以价格 $p$ 出售。**订得太少 → 失去销售机会。订得太多 → 卖不出去(当日结束时一文不值)。** 这种易腐库存的权衡适用于面包店、时尚行业、航空公司(超额预订)、季节性商品以及任何易腐商品。

## 1. 数学建模

- 随机需求 $D$,分布为 $F$
- 订货量 $Q$(在观测到 $D$ 之前选择)
- 成本 $c$,售价 $p$,残值 $s$(通常为 0)

**单次实现的利润:**
$$\pi(Q,D) = p \min(Q,D) + s(Q-D)^+ - cQ$$

当 $s = 0$ 时:$\pi = p \min(Q,D) - cQ$。

## 2. 两种等价的表述方式

### 基于利润
$$\max_Q E[p \min(Q,D) - cQ]$$

### 基于成本(缺货成本 & 过剩成本)
$$\min_Q E[c_u(D-Q)^+ + c_o(Q-D)^+]$$
- $c_u = p - c$(缺货成本 —— 每单位未满足需求的机会成本)
- $c_o = c - s$(过剩成本 —— 每单位未售出商品的净成本)

为何等价:利润 $= (p-c) E[D] - E[c_u(D-Q)^+ + c_o(Q-D)^+]$。第一项关于 $Q$ 是常数。

## 3. 临界分位数 —— 推导

### 边际分析
从 $Q$ 增加到 $Q+1$:
- 若 $D > Q$:获得 $c_u$(概率为 $P(D > Q)$)
- 若 $D \le Q$:损失 $c_o$(概率为 $P(D \le Q)$)

令边际收益 = 0:
$$c_u(1 - F(Q^*)) = c_o F(Q^*)$$
$$F(Q^*) = \frac{c_u}{c_u + c_o}$$

**基于利润的等价形式(当 $s = 0$ 时):** $F(Q^*) = (p-c)/p$。

### 微积分推导
$G(Q) = pE[\min(Q,D)] - cQ$。求导:
$$G'(Q) = p(1 - F(Q)) - c = 0 \Rightarrow F(Q^*) = \frac{p-c}{p}$$

**直觉理解:** 希望缺货概率 = $c/p$。利润率薄(高 $c/p$)→ 少备货。

## 4. 分布 —— CDF 反演

### 均匀分布 $U[a,b]$
$F(x) = (x-a)/(b-a)$,$Q^* = a + \alpha(b-a)$。

**例:** $U[20,80]$,$p=12, c=4, s=2$ → $c_u=8, c_o=2, \alpha=0.8$。$Q^* = 20 + 0.8(60) = 68$。

### 三角分布 Tr$[a, b, c]$(下限,众数,上限)
**PDF:**
$$f(x) = \begin{cases} \frac{2(x-a)}{(b-a)(c-a)} & a \le x \le b \\ \frac{2(c-x)}{(c-b)(c-a)} & b < x \le c \end{cases}$$

**CDF:**
$$F(x) = \begin{cases} \frac{(x-a)^2}{(b-a)(c-a)} & x \le b \\ 1 - \frac{(c-x)^2}{(c-b)(c-a)} & x > b \end{cases}$$

**反函数:**
- 若 $\alpha \le F(b)$:$Q^* = a + \sqrt{\alpha(b-a)(c-a)}$
- 若 $\alpha > F(b)$:$Q^* = c - \sqrt{(1-\alpha)(c-b)(c-a)}$

**例:** Tr[100, 150, 300],$p=4, c=1, s=0 \Rightarrow \alpha = 0.75$。$F(150) = 50/200 = 0.25$。由于 $0.75 > 0.25$,使用上分支:
$$Q^* = 300 - \sqrt{0.25 \cdot 150 \cdot 200} = 300 - 86.6 = 213$$

### 正态分布 $N(\mu, \sigma^2)$
$Q^* = \mu + z_\alpha \cdot \sigma$。当 $\alpha < 0.5$ 时 $z_\alpha < 0$。

标准 $z$ 值:$z_{0.5}=0, z_{0.75}=0.674, z_{0.9}=1.282, z_{0.95}=1.645$。

**例:** $N(500, 100^2), p=100, c=60, \alpha=0.4, z_{0.4}=-0.253$。$Q^* = 500 - 25 = 475$。

### 指数分布 Exp($\lambda$)
$F(x) = 1 - e^{-\lambda x}$,$Q^* = -\ln(1-\alpha)/\lambda$。

## 6. 样本均值(经验)报童问题

给定历史需求 $d_1, \ldots, d_n$:
1. 计算 $\alpha = (p-c)/p$
2. 对需求排序
3. $Q^* = d_{(\lceil n\alpha \rceil)}$(第 $\lceil n\alpha \rceil$ 小的值)

**例:** 10 个需求 [45,52,48,61,55,49,58,51,46,60]。$\alpha=0.8$,$\lceil 8 \rceil = 8$。排序后:[45,46,48,49,51,52,55,58,60,61]。$Q^* = 58$。

## 7. 背景变量报童问题 —— 总览

需求依赖于特征 $X$(天气、日期、地点)。希望得到 $Q^*(X)$ —— 作为背景变量函数的订货量。

两种方法:
- **回归树**(非参数方法,将特征空间划分为叶节点)
- **线性回归**(参数方法,假设 $D|X \sim$ 正态分布)

## 8. 回归树方法

1. 在训练数据 $(x_i, d_i)$ 上拟合回归树。
2. 对每个叶节点 $\ell$:收集 $\ell$ 中样本的需求,计算 $\alpha$-分位数。
3. 新样本 $x$ → 沿树下落找到叶节点 → 使用该叶节点的 $Q^*$。

## 9. 线性回归方法

假设 $D | X \sim N(\beta_0 + \beta_1 X_1 + \ldots + \beta_k X_k, \sigma^2)$。

1. 拟合线性回归 → 得到 $\hat\beta$ 和 $\hat\sigma$。
2. 对新样本 $x$:$\hat\mu(x) = \hat\beta_0 + \sum \hat\beta_i x_i$
3. $Q^*(x) = \hat\mu(x) + z_\alpha \cdot \hat\sigma$

**例:** $\hat\mu = 50 + 2T - 15W$,$\hat\sigma = 20$,$\alpha = 0.4$。明天 $T=85, W=0$ → $\hat\mu = 220$。$z_{0.4} = -0.253$。$Q^* = 220 - 5 = 215$。

## 11. 期望利润计算

$$E[\pi(Q^*, D)] = pE[\min(Q^*, D)] - cQ^*$$

其中:
$$E[\min(Q^*, D)] = \int_0^{Q^*} x f(x) dx + Q^* [1 - F(Q^*)]$$

替代形式:$G(Q^*) = (p-c)Q^* - p \int_0^{Q^*} F(x) dx$。

**详细例题(咖啡,均匀分布):** $Q^* = 68$,$E[\pi^*] = $352。完整利润:
$$\pi^* = pE[D] - E[\text{损失}] = 8 \cdot 50 - 48 = 400 - 48 = 352$$

## 12. 分步模板

### (a) 已知分布
1. 计算 $\alpha = (p-c)/p$ 或 $c_u/(c_u+c_o)$
2. 反演 CDF:$Q^* = F^{-1}(\alpha)$
3. (可选)计算期望利润

### (b) 样本数据
1. 计算 $\alpha$
2. 对需求排序
3. $Q^* = $ 第 ⌈$n\alpha$⌉ 小的值

### (c) 回归树
1. 将 $x$ 沿树下落找到叶节点 $\ell$
2. $Q^*(x) = $ $\ell$ 中需求的经验 $\alpha$-分位数

### (d) 线性回归
1. 拟合线性模型,得到 $\hat\beta, \hat\sigma$
2. 预测均值 $\hat\mu(x)$
3. $Q^*(x) = \hat\mu(x) + z_\alpha \hat\sigma$

## 13. 常见错误 ⚠️

1. **分位数方向错误。** $F(Q^*) = \alpha$(CDF),而**不是** $P(D > Q^*) = \alpha$。
2. **忘记残值。** 若 $s > 0$:$c_o = c - s$(不是只有 $c$)。
3. **二次方程根选错。** 三角分布:$Q^* = 300 \pm 86.6$ —— 选 $[a, c]$ 范围内的根,即 213。
4. **使用 $\lfloor n\alpha \rfloor$ 而不是 ⌈$n\alpha$⌉** 计算经验分位数。
5. **忘记 $\alpha < 0.5$ 时 $z_\alpha < 0$。** 缺货成本低时应订货量低于均值。
6. **将均值作为 $Q^*$。** 仅在 $\alpha = 0.5$ 时正确。
7. **回归树过拟合。** 使用 max_depth ≤ 4。
8. **三角分布 CDF 分支错误。** 先比较 $\alpha$ 与 $F(b)$。

### 关键公式(反复强调)

> *"有一个公式可以找到最优 Q。**无论**你的分布是什么或数据集是什么。"*

$$Q^* = F^{-1}\left(\frac{p-c}{p}\right) \quad \Leftrightarrow \quad F(Q^*) = \frac{p-c}{p}$$

> *"这非常容易计算。"*

**配套关系:** $P(\text{失去销售}) = c/p$,与分布无关。

### 关键澄清

**缺货成本 vs 过剩成本(精确的美元金额):**
- **过剩($Q > D$):** 支付了 $Q$ 单位的成本,但只卖出 $D$。每多 1 单位损失 $c$。
- **缺货($Q < D$):** 需求超过供应。每少 1 单位损失 $p - c$(放弃的利润,**不是**价格)。

> *"它会增加 p。你多卖一单位,但因为你需要多备一单位库存,所以损失 c。这就是为什么是 p 减 c。"*

**为什么 CDF 反演有效(边际逻辑):**
在最优处,边际收益 = 边际成本:
- 上行:$p \cdot P(D > Q)$(需求超过库存时的收入)
- 下行:$c \cdot 1$(多 1 单位的成本)
- 令其相等:$p(1 - F(Q^*)) = c \Rightarrow F(Q^*) = (p-c)/p$。

**连续 vs 离散需求:**
> *"我们假设 D 是连续随机变量……这其实让分析变得很容易。你会看到我们要处理一些积分。"* 对于离散数据,使用经验分位数(`np.quantile`)。

### 销售 vs 需求(关键)

> *"关键在于销售与需求之间的**差别**。"*

- 需求 $D$:顾客想要的数量。
- 销售 = $\min(Q, D)$:实际可以卖出的数量。
- 若 $Q = 80, D = 100$:卖出 80,损失 20 的机会成本。

### 橙汁案例 —— 关键洞察

Domenic's Spider Foods 连锁店,83 家门店,11 种橙汁产品。人口统计:60岁以上比例、大学比例、收入等。

**不同门店有**不同的**需求分布** → 背景变量报童问题很重要。

**回归树(max_depth=2):** 先按教育 < 0.449 划分,再按收入 < 0.87 划分。叶节点均值:16k、11.5k、38k、26k 单位。

> ❗ ***但这些叶节点均值不是最优订货量。***

> *"那其实不是我们的订货量应该取的值。"*

回归树的叶节点**值**最小化的是平方误差(预测均值)。报童问题需要的是每个叶节点中需求的 $(p-c)/p$ **分位数**,而不是均值。

**步骤:** 用叶节点中需求的经验分位数替换树叶节点**值**。

### 考试推导策略(背下这 6 步)

1. 写出利润:$\pi(Q) = p \min(Q, D) - cQ$
2. 取期望:$E[\pi] = pE[\min(Q,D)] - cQ$
3. 拆分:$E[\min] = \int_0^Q x f(x)dx + Q \cdot P(D > Q)$
4. 求导(微积分基本定理):$\frac{d}{dQ} \int_0^Q xf(x)dx = Q f(Q)$
5. 令导数 = 0:$p(1 - F(Q^*)) = c$
6. 求解:$F(Q^*) = (p-c)/p \Rightarrow Q^* = F^{-1}((p-c)/p)$

### 警告

- **销售 ≠ 需求。** 若 $Q < D$:销售 = $Q$,**不是** $D$。
- **点预测 ≠ 最优 Q。** 回归均值是为 MSE 服务;报童问题需要**分位数**。
- **易腐性假设** 仅在未售出 = 零残值时成立。可重用的库存不适用。

### 背景变量报童问题 —— 完整流程

1. 在训练数据 $(X_i, D_i)$ 上训练预测模型(回归树 / 线性回归 / k-NN)。
2. 对每个"区域"(树叶节点),计算该区域中需求的 $(p-c)/p$ **经验分位数**。
3. 对新背景变量 $x$:找出 $x$ 落入哪个区域 → 使用该分位数作为 $Q^*(x)$。
