线性规划对偶

概念公式说明
弱对偶$c^T x \le p^T b$两者都必须可行。对于求极小的原问题,原问题值 ≥ 对偶问题值。
强对偶$c^T x^* = p^{*T} b$在最优解处,两者相等。
互补松弛$x_j^* > 0 \Rightarrow$ 第 $j$ 个对偶约束紧同时:约束松弛量 > 0 $\Rightarrow p_i = 0$。
影子价格的应用$\Delta\text{obj} \approx p_i^* \cdot \Delta b_i$仅适用于微小变化。
“多买 X”规则当市场价格 < $p_i^*$ 时购买每单位净收益 = $p_i^* - \text{成本}$。

原问题→对偶问题转换(求极小的原问题)

原问题约束对偶变量
$a_i^T x \ge b_i$$p_i \ge 0$
$a_i^T x \le b_i$$p_i \le 0$
$a_i^T x = b_i$$p_i$ 自由
原问题变量对偶约束
$x_j \ge 0$$p^T A_j \le c_j$
$x_j \le 0$$p^T A_j \ge c_j$
$x_j$ 自由$p^T A_j = c_j$

对于求极大的原问题:从右往左读 — 符号反转。

报童模型

概念公式
缺货成本$c_u = p - c$
过剩成本$c_o = c - s$($s$ = 残值)
临界分位数$\alpha = c_u/(c_u+c_o) = (p-c)/p$(当 $s=0$ 时)
最优 $Q^*$$Q^* = F^{-1}(\alpha)$
均匀分布 $U[a,b]$$Q^* = a + \alpha(b-a)$
正态分布 $N(\mu, \sigma^2)$$Q^* = \mu + z_\alpha \sigma$
经验分布(样本)$Q^* = d_{(\lceil n\alpha \rceil)}$(升序排序)
线性回归情境化$Q^*(x) = \hat\mu(x) + z_\alpha \hat\sigma$
决策树情境化$Q^*(x) = $ 叶节点需求的经验 $\alpha$ 分位数

三角分布 Tr$[a, b, c]$

概率密度函数(PDF):

$$f(x) = \begin{cases} \frac{2(x-a)}{(b-a)(c-a)} & a \le x \le b \\ \frac{2(c-x)}{(c-b)(c-a)} & b < x \le c \end{cases}$$

累积分布函数(CDF):

$$F(x) = \begin{cases} \frac{(x-a)^2}{(b-a)(c-a)} & x \le b \\ 1 - \frac{(c-x)^2}{(c-b)(c-a)} & x > b \end{cases}$$

反函数:

  • 若 $\alpha \le F(b)$:$Q^* = a + \sqrt{\alpha(b-a)(c-a)}$
  • 若 $\alpha > F(b)$:$Q^* = c - \sqrt{(1-\alpha)(c-b)(c-a)}$

均值:$(a + b + c)/3$。

收益管理

概念公式
预期到达量(伯努利)$E[D_j] = (T-t+1) p_j$
预期到达量(泊松)$E[N] = \lambda_j (T-t)$
容量 LP$\max \sum r_j x_j$ s.t. $\sum x_j \le C$,$x_j \le E[D_j]$,$x \ge 0$
投标价格$\pi_\ell$ = 容量约束的影子价格
单航段规则当 $r_j \ge \pi_t(b)$ 时接受
多航段规则当 $r_i \ge \sum_{\ell \in S_i} \pi_\ell$ 时接受行程 $i$
LP 值 =预期收益的上界

商品组合

概念公式
选择概率(首选)$P(i|S) = \sum_{k: i = \arg\max_{j \in S \cup \{0\}} u_{k,j}} \lambda_k$
预期收益$R(S) = \sum_{i \in S} r_i P(i|S)$
定制化上界$UB_1 = \sum_k \lambda_k \max_{i: u_{k,i} > u_{k,0}} r_i$
MNL 选择概率$P(i|S) = e^{u_i} / (e^{u_0} + \sum_{j \in S} e^{u_j})$
LP 松弛去掉 $x, y$ 的 0-1 约束;LP 值 ≥ IP 最优值

定价(线性需求)

$$d_i(\mathbf{p}) = \alpha_i - \beta_i p_i + \sum_{j \ne i} \gamma_{i,j} p_j$$ $$R(\mathbf{p}) = \sum_i p_i d_i(\mathbf{p})$$
  • $\gamma_{i,j} > 0$:替代品;$\gamma_{i,j} < 0$:互补品
  • $\beta_i > 0$ 恒成立;自价格效应对收益的贡献为 $-\beta_i p_i^2$(负值)
  • 离散 IP:$x_{i,t}$ 为 0-1 变量,$\sum_t x_{i,t} = 1$,$y_{i,j,t_1,t_2}$ 需要两组联结约束

选址模型

模型目标函数变量关键约束
MCLP(最大覆盖)$\max \sum y_j$$x_i$,$y_j$$y_j \le \sum_{i \in N(j)} x_i$
p-中位$\min \sum d_{ij} z_{ij}$$x_j$,$z_{ij}$$\sum_j z_{ij} = 1$,$z_{ij} \le x_j$,$\sum x_j = p$
p-中心$\min r$$x_j$,$z_{ij}$,$r$$r \ge \sum d_{ij} z_{ij}$ + p-中位约束

建模强度:强形式 $z_{ij} \le x_j$(逐对约束)≫ 弱形式 $\sum_j z_{ij} \le m \cdot x_i$(聚合约束)。强形式给出更紧的 LP 松弛,求解速度更快。

网络流

概念公式
流量守恒在每个节点 $i$ 处:$\sum_{j: (j,i) \in E} x_{ji} + b_i = \sum_{j: (i,j) \in E} x_{ij}$
供需符号$b_i > 0$ 为供给;$b_i < 0$ 为需求;$b_i = 0$ 为转运
平衡$\sum_i b_i = 0$
最短路径设置$b_s = +1$,$b_t = -1$,其他 $= 0$
最大可靠度$c_{ij} = -\log r_{ij}$,然后求极小
分段成本低价段变量(有上限)+ 超额变量;两者均计入流量守恒
前 K 条路径添加 $\sum_{(i,j) \in E^*_k} x_{ij} \le L_k - 1$,然后重新求解
全幺模性纯网络流 LP ⇒ 最优解为整数(无需 IP)

旅行商问题(TSP)

约束公式
出度$\sum_{j \ne i} x_{ij} = 1 \quad \forall i$
入度$\sum_{j \ne i} x_{ji} = 1 \quad \forall i$
子集 $S$ 的子环消除约束(SEC)$\sum_{i \in S} \sum_{j \in S, j \ne i} x_{ij} \le |S| - 1$
割形式的 SEC$\sum_{i \in S, j \notin S} x_{ij} \ge 1$
曼哈顿距离$d_{ij} = |u_i - u_j| + |v_i - v_j|$
惰性回调在 MIPSOL 处检测子环,动态添加 SEC

通用附加约束模式

语言描述约束
若 $A$ 则 $B$$x_A \le x_B$
若 $A$ 则 $\{B_1,B_2,B_3\}$ 中至少一个$x_A \le x_{B_1}+x_{B_2}+x_{B_3}$
若 $A$ 且 $B$ 则 $C$$x_A + x_B - 1 \le x_C$
若 $A$ 且 $B$ 则 $C$ 且 $D$$x_A + x_B - 1 \le x_C$ 且 $x_A + x_B - 1 \le x_D$
若 $A$ 则非 $B$$x_A + x_B \le 1$
互斥$x_A + x_B \le 1$
至少选 $m$ 个,至多选 $M$ 个$m \le \sum x_i \le M$
恰好选 $p$ 个$\sum x_i = p$
容量限制(每个设施 $K$ 单位)$\sum_i z_{ij} \le K \cdot x_j$
条件概率 ≥ 20%$\sum_k \lambda_k y_{k,i} \ge 0.20 \cdot x_i$
